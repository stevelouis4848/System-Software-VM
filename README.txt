#Info

This C programs mimick the inner working of a ""
	virtual machine.An input file with the instructions
	are passed in and the program runs it and prints out the
	instructions, stacktrace and factorial output to three
	different files.


# How to run


- compile in Linux using gcc vm.c

- run the program with the input file
	./a.out "name of input file with instructions"
	
- The program will print the output to three files one 
	file named ""has the instructions the next file named
	"" has the stack trace and registers and the last file named "" has the
	registers that printed do to an instruction.
